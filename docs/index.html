Portugal Freelancer Tax Data 2026

Free, machine-readable JSON datasets for Portuguese freelancer tax rules. Built for digital nomads, D8 visa holders, and expat freelancers.

This open-source project provides structured tax data for anyone building tools, spreadsheets, or calculators for freelancers in Portugal. All data is sourced from the Autoridade Tributaria (Portuguese Tax Authority) and updated for 2026.
